# Scripts SQL - VamonosPues

Estructura de scripts organizados por tipo:

## 📁 Carpetas

- **DDL/** - Definición de estructura (tablas, esquemas)
- **Constraints/** - Restricciones y relaciones
- **Indices/** - Índices para optimización
- **DML/** - Datos (inserciones, populate)

## 🔄 Orden de ejecución

1. `DDL/` - Ejecutar primero
2. `Constraints/` - Luego
3. `Indices/` - Después
4. `DML/` - Finalmente (para insertar datos)
